# Scientific Programming in Julia

This repository contains solutions to homework assignments from the **Scientific Programming in Julia** course at the Czech Technical University in Prague.

## Repository Contents

### Matrix Polynomials

Extension of polynomial evaluation to square matrices and creation of a circular graph adjacency matrix.

### Predator–Prey Simulation

Implementation of agent-counting methods for animals, plants, collections of agents, and complete ecosystem simulations.

### Food Search and Closures

Implementation of food selection for predator–prey agents and a callback that executes a function every specified number of calls.

### Package Testing

Unit tests for interactions between sheep, wolves, grass, and the simulated ecosystem.

### Root-Finding Optimization

Performance optimization and type-stability improvements for bisection, Newton, and secant root-finding methods.

### Expression Analysis

AST traversal for extracting unique variables from Julia expressions while handling assignments and function calls.

### Ecosystem Macro

A custom macro for creating simulated worlds containing configurable numbers of plants and animals.

### Automatic Differentiation

Extension of a scalar reverse-mode automatic differentiation system with rules for division and arithmetic with constants.

### Parallel Convolution

A multithreaded implementation of discrete one-dimensional convolution using Julia threads.

### Runge–Kutta ODE Solver

Implementation of the second-order Runge–Kutta method for solving ordinary differential equations, demonstrated on the Lotka–Volterra model.

## Main Topics

* Julia type system and multiple dispatch
* Design patterns and closures
* Package development and unit testing
* Benchmarking and performance optimization
* Metaprogramming and macros
* Automatic differentiation
* Multithreading
* Numerical methods
